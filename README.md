## Sobre o Projeto

Este projeto é um desafio do módulo 01 - Node.js do curso Go Stack;

### Feito com

- [Javascript]
- [NodeJS]

## Começando

Após ter clonado o projeto rodar o seguinte comando na pasta do projeto:

```sh
yarn
```

e depois para rodar em modo desenvolvimento:

```sh
yarn dev
```

e para testar:

```sh
yarn test
```
